# bootstrappedMyApachePHP
Modernizing Apache's File Indexer UI using Bootstrap 5.0 and PHP. bootstrappedMyApachePHP is a responsiveness focussed skinning project for our favorite file servers.

### Features
* Modernized Look - Using Bootstrap v5.0 and Google Material Icons
* Responsive Interface - Server loads everything on visit, as a result, smooth experience during navigation.

### Performance
* Server sends everything to the frontend it is allowed to see. 
* Click response is close to 0ms even if you use it over 2G Networks.
* Initial loading time is still under work. ;)
