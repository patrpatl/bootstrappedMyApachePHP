# bootstrappedMyApachePHP
Modernizing Apache's File Indexer UI using Bootstrap 5.0 and PHP.

# Performance
Server sends everything to the frontend it is allowed to see. 
Click response is close to 0ms even if you use 2G Networks.
Initial loading time is still under work. ;)
